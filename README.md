To become a tasking tool similar to Jira.

Frontend:
    SvelteKit web app consisting of:
        A home page linking to all other pages
        A "task creation" page, where a task can be created and added to the board.
        A "board" page consisting of columns: "To-Do", "In Progress", and "Done"

Backend:
    Postgresql database which stores all tasks, including the task name and description, as well as task status.
